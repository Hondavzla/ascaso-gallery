## Project: Ascaso Gallery Landing Page

### Stack
React + Vite, Tailwind CSS v3, GSAP, Three.js, Framer Motion

### Aesthetic Direction
Luxury editorial. Gagosian meets Miami sunlight.
- Display font: Cormorant Garamond
- Body font: DM Sans
- Colors: warm white #FAF9F6, near-black #0D0D0D, gold accent #C9A84C
- Motion: slow and cinematic — nothing fast or bouncy
- Large negative space, editorial layouts, artwork is always the focus

### Rules
- Never use Inter, Roboto, or Arial
- No purple gradients, no heavy drop shadows
- Mobile responsive from the start
- Every section must feel intentional, not templated

### Build Order
1. Hero — fullscreen GSAP entrance + Three.js background
2. Current Exhibition — large editorial card
3. Featured Artists — full-bleed portrait grid
4. Latest News — editorial card row
5. Newsletter — minimal inline form
6. Footer — Miami + Caracas, logo, links
